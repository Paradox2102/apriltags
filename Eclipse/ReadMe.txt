This folder contains a java app which allow you to view the Apriltags and control the camera.
This project uses the Eclipse IDE 2022.

A prebuilt ApriltagsViewer.jar file is provided for those who would rather not install Eclipse.
When running the program, you must supply the IP address of your Pi on the command line.
See 'ApriltagsViewer.cmd' for an example of how to run it.

1/4/2023:

A second java app was added which connects to the RoboRio and displays the robots position and
direction on the field in real time.

A prebuild PositionViewer.jar file is once again provided. Note the 'PositionViewer.cmd' file which
is an example of how to run it. Also note that the file 'field.jpg' must be present in the same folder.
