This folder contains all of the source code to build the Apriltag programs.

All of these programs are CodeBlocks projects. However, makefiles are provided for building the
projects without installing Codeblocks.

Note that this code is present on the Pi images for the respective camera and those images also contain
a pre-built version of the executable.

apriltagVisionBW - This is a project that creates the version which supports the black and white camera
apriltagVisionPiCamera - This is a project that creates the version wich supports the normal Pi camera
apriltagCommon - This is the common code which is shared between the two version.
